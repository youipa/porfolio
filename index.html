🚀 Projets réalisés

Voici un aperçu de quelques projets que j'ai pu réaliser au cours de cette première année, couvrant certains les aspects de la formation RT :

Configuration réseau d'entreprise (SAE12)

Conception et configuration d’un réseau d’entreprise avec plusieurs VLANs, routage inter-VLAN, plan d’adressage IP, et mise en place de services essentiels comme le DHCP. Ce projet m’a permis d’appliquer les bases pratiques de l’administration réseau en environnement professionnel.

Cisco Packet Tracer VLAN OSPF ACL

Creation site internet Peugeot (SAE14)

Développement d’un site web complet et responsive dédié à la marque Peugeot. Intégration de contenus multimédias, respect des normes HTML/CSS et accessibilité, mise en place d’une navigation fluide et d’un design moderne. Ce projet m’a permis de renforcer mes compétences en développement web front-end et en design adaptatif.

html css JavaScript

Application de suivi des données Pokémon (SAE15)

Conception et développement d’une application web interactive exploitant l’API Pokéfiche pour récupérer et afficher des informations détaillées sur les Pokémon. L’application permet d’accéder à leurs statistiques, évolutions, types et autres caractéristiques, avec une interface conviviale facilitant la navigation et la comparaison.

JavaScript API REST HTML/CSS Fetch API
